  - 7 個分類（About/Economy/Language/Lifestyle/Nature/Society）腳註數為零

  - 結構：母親難產死亡開場 → 捐款編年（含精確金額和日期）→ 國際認可 → 身體崩壞 → 收尾回扣母親
