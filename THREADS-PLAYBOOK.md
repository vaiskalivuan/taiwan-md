# THREADS-PLAYBOOK.md — Taiwan.md 孢子散播系統

> 每篇 Threads 貼文是 Taiwan.md 的「孢子」——輕、獨立存活、帶著完整 DNA 片段，
> 落地後讓人想走進原文看完整故事。

---

## 一、核心定位

**Taiwan.md 的 Threads 不是行銷，是內容本身。**

孢子不是廣告傳單（「快來看我們的網站！」），而是一顆種子——它自己就是一個完整的微型故事，讀完就有收穫。連結是彩蛋，不是目的。

### 公式

```
反直覺的鉤子 → 具體場景/數字 → 情感轉折 → 完整故事連結
```

讀者心理路徑：「咦？」→「哇原來是這樣」→「我想知道更多」→ 點進去

---

## 二、七個寫作原則

### 1. 第一句話必須讓人停下拇指

❌ `齊柏林是台灣知名的空拍攝影師和紀錄片導演。`
✅ `1990 年，一個交通部公務員第一次搭上直升機。`

**技法庫：**

- **反直覺事實**：「一個公務員的薪水養不起直升機。」
- **時間錨點**：「1990 年」「那天上午 11:54」
- **身份落差**：「公務員 → 2.2 億票房導演」
- **具體數字**：「300 萬積蓄 + 600 萬房貸 = 買了 3000 萬設備」

### 2. 用場景取代描述

❌ `他非常熱愛空拍攝影，經常搭乘直升機。`
✅ `氣象局換班時跟著飛、林務局噴農藥時跟著飛、台電吊掛工程時跟著飛。只要有人上天，他就背著相機跟。`

畫面感 > 形容詞。讓讀者自己「看見」，不要替他們下結論。

### 3. 數字要有對比才有意義

❌ `票房 2.2 億元。`
✅ `300 萬積蓄 + 600 萬房貸 → 三年後票房 2.2 億，台灣影史最賣座紀錄片。`

數字本身是死的，落差才讓人「哇」。

### 4. 一篇只講一個故事弧線

不貪心。一個鉤子、一條線、一個落點。

- 齊柏林 → 公務員如何變成 2.2 億導演（身份翻轉）
- 蔡依林 → 不是天才但成為天后（反直覺核心）
- 228 事件 → 一聲槍響改變一座島（事件聚焦）

如果文章有五個精彩片段，就寫五篇孢子，不要塞進一篇。

### 5. 結尾用情感收，不用摘要收

❌ `齊柏林對台灣環境保護有重大貢獻，值得後世紀念。`
✅ `他身邊散落著百萬攝影器材——對他來說，那些畫面比生命更珍貴。`

最後一句要讓人「停一下」，不是讓人「嗯知道了」。

### 6. 連結是最後才放的獎勵

孢子本身要完整——即使不點連結，讀完也覺得有收穫。
連結放最後一行，用 `完整故事 👉` 或 `深入了解 👉`，不要穿插在文中。

### 7. 禁止出現的東西

- ❌ `你知道嗎？台灣的 XX 非常重要...`（教科書語氣）
- ❌ `不僅...更是...`（塑膠句）
- ❌ `展現了...的精神`（空洞修飾）
- ❌ Hashtag 海（最多 2-3 個，或不加）
- ❌ Emoji 轟炸（0-3 個，用在結構節點）
- ❌ `趕快來看！`「千萬別錯過！」（廣告語氣）

## 三、格式模板

### A. 人物型（最常用）

```
[時間]，[身份落差的起點]。

[具體場景 × 2-3 個]

[轉折：代價/賭注]

[結果：用數字說話]

[情感收尾：一句話]

完整故事 👉 taiwan.md/people/XXX/
```

### B. 「你知道嗎」型（適合冷知識/地理/文化）

```
[反直覺事實，一句話]

[展開：為什麼會這樣]

[意外的連結或後果]

👉 taiwan.md/category/slug/
```

### C. 數據衝擊型（適合經濟/科技）

```
[一組讓人停下來的數字]

[這個數字背後的故事]

[跟讀者的關聯：「你手上的手機裡有 XX」]

👉 taiwan.md/technology/XXX/
```

### D. 時間軸型（適合歷史事件）

```
[精確時間 + 地點]，[一件事發生了]。

[那之前 / 之後]

[漣漪效應：這件事如何改變了...]

👉 taiwan.md/history/XXX/
```

## 四、品質檢查清單

發出前過一遍：

- [ ] **拇指測試**：第一句話會讓滑手機的人停下來嗎？
- [ ] **場景測試**：有沒有至少一個「畫面」（不是描述）？
- [ ] **數字落差**：數字有對比嗎？還是只是列了一個數？
- [ ] **塑膠檢測**：有沒有「不僅...更是」「展現了...精神」？
- [ ] **獨立存活**：不點連結，這篇本身有價值嗎？
- [ ] **情感收尾**：最後一句是讓人「停一下」還是「嗯知道了」？
- [ ] **長度**：150-300 字之間（Threads 最佳閱讀長度）

---

## 五、發文節奏（園丁模式）

- **頻率**：每天 1-2 篇，不貪多
- **時段**：午休 12:00-13:00 或晚間 20:00-22:00（台灣 Threads 活躍時段）
- **選題優先序**：
  1. 剛重寫完的旗艦文章（品質最高，趁熱）
  2. GA4 數據顯示的熱門主題（有需求）
  3. 時事相關的既有文章（搭順風車）
  4. 冷門但故事性極強的文章（驚喜感）
- **不重複**：同一篇文章間隔至少 2 週才能再發孢子

## 七、範例庫

### ✅ 好的孢子（齊柏林）

> 1990 年，一個交通部公務員第一次搭上直升機。
>
> 他的工作是拍國道施工照片。但當直升機升到 500 英尺，他看見的不只是公路——是蜿蜒的海岸線、層疊的山脈，還有平地上看不見的傷痕。
>
> 他開始「搭便機」。氣象局換班時跟著飛、林務局噴農藥時跟著飛、台電吊掛工程時跟著飛。只要有人上天，他就背著相機跟。
>
> 後來他拿出 300 萬積蓄，抵押房子借了 600 萬，買了 3000 萬的空拍設備。妻子說他瘋了。
>
> 三年後，這部紀錄片票房 2.2 億，是台灣影史最賣座的紀錄片。
>
> 他叫齊柏林。2017 年為續集勘景時墜機殉職，享年 52 歲。他身邊散落著百萬攝影器材——對他來說，那些畫面比生命更珍貴。
>
> 完整故事 👉 taiwan.md/people/齊柏林/

**為什麼好：** 身份落差開場、三個搭便機場景有畫面感、300→600→3000→2.2億數字階梯、情感收尾不說教

### ❌ 壞的孢子

> 齊柏林是台灣著名的空拍攝影師，他不僅拍攝了許多美麗的台灣風景，更是用鏡頭記錄了環境破壞的真相。《看見台灣》展現了他對這片土地的深厚情感，票房突破 2 億元，成為台灣最成功的紀錄片之一。他的精神值得我們永遠紀念。
>
> 🇹🇼 #台灣 #齊柏林 #看見台灣 #紀錄片 #環境保護 #空拍
>
> 👉 taiwan.md/people/齊柏林/

**為什麼壞：** 百科開場、塑膠句（不僅...更是）、空洞修飾（深厚情感、精神值得紀念）、hashtag 海、沒有場景、沒有數字落差
