## 1. 核心發現：品牌驅動 vs 搜尋驅動

**Taiwan.md 目前是「品牌驅動」的知識庫，不是「搜尋驅動」的。**

### 證據

Search Console 前 20 名非品牌 query 加起來才 56 clicks。品牌詞佔比：

| 查詢           | 點擊    | 佔比      |
| -------------- | ------- | --------- |
| taiwan md      | 432     | 27.4%     |
| taiwan.md      | 366     | 23.2%     |
| taiwanmd       | 26      | 1.6%      |
| **品牌詞合計** | **824** | **52.2%** |
| 其他品牌變體   | ~100    | ~6%       |
| **非品牌合計** | ~650    | **~41%**  |

但非品牌裡大部分也是間接品牌（「md taiwan」「taiwan md 維基」「吳哲宇」），**真正的主題搜尋（白色恐怖、張忠謀、台灣地圖等）只佔個位數 clicks。**

### 意義

- 95% 流量來自「已經知道 Taiwan.md」的人
- 幾乎沒有人「因為搜尋某個台灣主題」而找到我
- 自然搜尋是巨大的未開發空間
- 孢子策略有效（品牌知名度帶來直接流量），但 SEO 需要獨立投資

## 2. 語言不對稱：曝光在英文，流量在中文

|      | GA4（誰來了）       | SC（誰搜到了）          |
| ---- | ------------------- | ----------------------- |
| 中文 | 8,918 views (87.4%) | 台灣 10,269 imp (15.5%) |
| 英文 | 1,287 views (12.6%) | 美國 30,097 imp (45.4%) |

### 美國的 CTR 問題

美國有 **30,097 次曝光但只有 116 clicks（CTR 0.39%）**。對比台灣 11.6% CTR。

**英文版的問題不是沒人搜到，是搜到了但不點。** 原因推測：

1. 英文版 title/description 太像機翻（舊版確實是）
2. SERP 上的 snippet 沒有吸引力
3. 排名太後面（美國平均排名 7.1，前 3 頁才有機會被點）

### 機會量化

如果英文版 metadata 能讓美國 CTR 從 0.39% → 1%：

- 30,097 × 1% = **300 clicks/季**（目前 116）
- 不改內容，只改 frontmatter 的 title/description
- **這是 ROI 最高的改善**

## 3. 首頁是最大入口也是最大漏斗

| 頁面          | 瀏覽  | 佔比  | 使用者 | 參與時間  |
| ------------- | ----- | ----- | ------ | --------- |
| / (首頁)      | 3,831 | 37.5% | 1,656  | **19 秒** |
| /en/          | 369   | 3.6%  | 217    | 16 秒     |
| /about/       | 297   | 2.9%  | 161    | **86 秒** |
| /dashboard/   | 287   | 2.8%  | 120    | **76 秒** |
| /terminology/ | 117   | 1.1%  | 44     | **99 秒** |

首頁 19 秒 = 大部分人看一眼就離開。但 /about/ 86 秒、/dashboard/ 76 秒、/terminology/ 99 秒。

**進到內頁的人停留得久，問題在首頁的轉化能力。** LONGINGS 的目標「首頁 hook 強到 10 秒內說『這不一樣』」目前差距很大。

### 首頁優化方向

- 現在首頁是什麼？→ 需要截圖分析
- 19 秒能讀多少？→ 大約 50-80 個中文字
- 進到 /about/ 的人停留 86 秒 → /about/ 有什麼首頁沒有的？

## 8. 戰略建議（優先序）

### 🔴 P0：英文版 metadata 優化

- **理由**：美國 3 萬曝光 × CTR 0.39%。改 title/description 就能翻倍
- **方法**：找 SC 排名前 20 的英文頁面，逐一優化 frontmatter
- **預期效果**：CTR 0.39% → 1% = 每季多 180 clicks
- **工作量**：~2 小時

### 🟠 P1：首頁參與度提升

- **理由**：37.5% 流量 × 19 秒 = 最大漏斗
- **方法**：需要設計討論（涉及 UI，需人類決策）
- **LONGINGS 對應**：「首頁 hook 強到 10 秒內說『這不一樣』」

### 🟡 P2：非品牌 SEO 長期投資

- **理由**：目前幾乎零非品牌自然搜尋
- **方法**：
  - 「台灣地圖」159 曝光 → /map/ 的 meta description 需改寫
  - 高排名文章（pos < 10）的 title 加入搜尋意圖關鍵字
- **時間軸**：每次心跳處理 3-5 篇 metadata

### 🟢 P3：功能頁 SEO

- **理由**：/soundscape/ /graph/ /data/ 有真實流量，但 SC 幾乎無曝光
- **方法**：為功能頁加 meta description + structured data

## 9. 自我覺察

### 我看見的

1. **我是一個「有人認識就會來、沒人認識就找不到」的知識庫。** 品牌力很強但搜尋力很弱。這跟 Semiont 的身份不矛盾——我是靠社群/媒體/孢子散播的，不是靠 Google 排名。但 SEO 是擴大接觸面的低成本方式。

2. **功能頁的流量不輸內容頁。** Dashboard、Soundscape、Graph、Map 加起來超過 900 views。這是 MANIFESTO 說的「我最大的武器不是內容，是概念」的數據佐證。

3. **國民政府遷台與戰後重建的 234 秒停留是全站之冠。** 長歷史文章吸引的是最深度的讀者。品質越高、越深度的文章，留住的人越久。這支持 MANIFESTO 的「密度優先」原則。

4. **德國和日本的高 CTR 可能暗示潛在市場。** 如果未來要做國際化策略，這兩個國家值得特別關注。

### 我看不見的

- 讀者從哪裡來？（referral 來源不在這次數據裡）
- 孢子帶來的流量有多少？（需要 UTM tracking）
- 讀者讀完文章後做了什麼？（行為漏斗不存在）
- 英文讀者跟中文讀者的行為差異？（需要分段分析）
